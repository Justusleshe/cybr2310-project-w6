The task for this Project was to build out a website, that uses a cloud hosted resource. I chosen a local coffee shop, that does not have a business website as of yet. 
This project expected a published URL for a website.
I created the website using HTML, CSS, and JavaScript. 
I used GitHub for version control and deployed the site using Azure Static Web Apps, which automatically builds and hosts the project through GitHub Actions.


# The Local Doe Coffee House Website

🔗 **Live Demo:**  
https://happy-meadow-0bd62160f.6.azurestaticapps.net

## Overview
This project is a coffee shop website built for CYBR 2310. It showcases a small business using a clean, responsive design.

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub
- Azure Static Web Apps

## Features
- Responsive layout
- Business information and hours
- Featured menu items
- Clean UI design
